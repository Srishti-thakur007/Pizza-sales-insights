# 🍕 Pizza Sales Dashboard (Power BI Project)

## 📌 Project Overview

This project focuses on analyzing pizza sales data to uncover meaningful insights about customer behavior, sales performance, and business trends.
The dashboard provides a complete view of sales, including daily trends, peak hours, category performance, and best/worst-selling products.

---

## 🚀 Tools & Technologies Used

* 📊 Power BI
* 🗄️ SQL
* 📁 Excel (CSV Data Source)
* 🔧 Power Query
* 🧠 DAX (Data Analysis Expressions)

---

## 📂 Project Workflow

### 🔹 1. Data Collection

* Imported raw pizza sales data in **CSV (Excel)** format

### 🔹 2. Data Storage

* Loaded data into **SQL Database**

### 🔹 3. SQL Analysis

Performed SQL queries for:

* ✅ KPI calculations (Total Revenue, Total Orders, etc.)
* 📈 Daily Trend Analysis
* 📊 Percentage of Sales by Category & Size
* 🔝 Top 5 Best Sellers
* 🔻 Bottom 5 Worst Sellers

### 🔹 4. Data Import to Power BI

* Connected **Power BI to SQL Database**
* Imported cleaned dataset

### 🔹 5. Data Cleaning

* Used **Power Query Editor** for transformation
* Fixed data types, removed inconsistencies

### 🔹 6. Data Modeling & Measures

* Created KPIs using **DAX**
* Built calculated columns (Hour, Trends, etc.)

### 🔹 7. Dashboard Development

Created interactive visuals:

* 📊 KPI Cards (Revenue, Orders, Quantity)
* 📈 Daily Trend of Orders
* 🍕 Sales by Category
* 📏 Sales by Size
* ⏰ Peak Hour Analysis
* 🔝 Best & Worst Selling Pizzas

### 🔹 8. Dashboard Enhancement

* 🎛️ Added slicers (filters)
* 🔘 Navigation buttons for better user experience
* 🧾 Added insights & annotations

---

## 📊 Key Insights

* 📈 Orders are higher at the beginning of the week (Mon–Wed)
* 📉 Sales gradually decline towards the weekend
* ⏰ Peak hours show highest customer activity
* 🍕 Certain categories contribute more to overall revenue
* ⭐ Best-selling pizzas drive majority of sales
